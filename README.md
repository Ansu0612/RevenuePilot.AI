# RevenuePilot.AI

An AI-powered SaaS Revenue Intelligence Platform that centralizes CRM, marketing, subscription, billing, and product usage data into a unified analytics ecosystem.

## Objective

Build an industry-level portfolio project that demonstrates product thinking, database design, ETL automation, business intelligence, AI integration, and executive reporting.

## Technology Stack

* MySQL
* Python
* Power BI
* Claude AI
* Git & GitHub
* Notion
* Jupyter Notebook

## Project Status

🚧 Currently under development.

## Repository Structure

* `/database` – Database schema and SQL scripts
* `/data` – Sample and generated datasets
* `/docs` – Project documentation
* `/etl` – Data extraction, transformation, and loading scripts
* `/notebook` – Jupyter notebooks
* `/powerbi` – Power BI dashboards
* `/prompts` – Claude AI prompts
* `/tests` – Validation scripts
* `/assets` – Images and diagrams
